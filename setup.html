🚀 Guia de Instalação – Flowtask

1 Criar Banco de Dados MySQL

Acesse seu PHP MyAdmin ou terminal MySQL e execute:

CREATE DATABASE flowtask;
USE flowtask;

Ou se preferir usando a linha de comando:

mysql -u root -p
CREATE DATABASE flowtask;
EXIT;

2 Configurar arquivo backend/config.php

Edite o arquivo backend/config.php com suas credenciais MySQL:

define('DB_HOST', 'localhost');
define('DB_USER', 'seu_usuario');  // Altere conforme sua configuração
define('DB_PASS', 'sua_senha');    // Altere conforme sua configuração
define('DB_NAME', 'flowtask');

Nota: Se não tem senha MySQL, deixe DB_PASS vazio.

3 Iniciar Servidor Local

Use o PHP built-in server ou um servidor Apache/Nginx.

Opção 1 - PHP Built-in:

cd /caminho/para/flowtask
php -S localhost:8000

Opção 2 - XAMPP/WAMP: Coloque a pasta dentro de htdocs ou www e acesse via http://localhost/flowtask

4 Acessar o Sistema

Abra seu navegador e acesse:

http://localhost:8000

Ou se estiver usando XAMPP:

http://localhost/flowtask

Você será redirecionado para a tela inicial (landing page).

📋 Fluxo de Uso

  • Acesse a landing page (página inicial)
  • Clique em "Entrar" para ir para login
  • Se não tem conta, clique em "Cadastre-se"
  • Após cadastro, faça login com seu email e senha
  • Você será redirecionado para o dashboard principal
  • Use o botão de tema (🌙/☀️) para alternar entre modo claro e escuro

✨ Funcionalidades Adicionadas

  • Landing Page: Página inicial com descrição do sistema
  • Login: Autenticação segura com hash de senha
  • Cadastro: Registro de novos usuários com validação
  • Tema Global: Modo claro/escuro sincronizado em todas as páginas
  • Banco de Dados: Persistência de usuários, projetos e tarefas no MySQL
  • Segurança: Senhas com hash BCRYPT e validação de email

❓ Solução de Problemas

Erro: "Erro na conexão com o banco de dados"

  • Verifique se MySQL está rodando
  • Confira as credenciais em backend/config.php
  • Verifique se o banco de dados flowtask foi criado

Erro: "Email ou senha incorretos"

  • Verifique se você já cadastrou sua conta
  • Certifique-se de que email e senha estão corretos
